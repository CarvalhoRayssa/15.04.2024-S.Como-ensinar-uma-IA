1-A pergunta:
O que é Tecnologia e inovação?

2-Definições:
Defina o que é tecnologia.
Tecnologia é a técnica para aprender algo, são inovações!

Defina o que é inovação.
Inovação é um upgrade uma atualização de algo antigo, é mais moderno!

Defina o que é moderno.
Moderno é algo que acreditamos ser para frente, do futuro!

Defina o que é atualização.
É o atual de sua melhor forma, o seu melhor no momento!

Defina melhor.
Melhor é quando algo ou alguém supera a si mesmo!

Defina forma.
É o que alguém pode se tornar, uma forma é como uma melhor versão de si!

Defina versão.
Versão seria tipo uma fase!

3-Resumo:
Tecnologia e inovação, são técnicas e atualizações, são modernos e nos fazem acreditar que é algo do futuro
Eles são importantes, fazem com que as ´pessoas queiram melhorar e superar a si mesmos, como uma melhor versão de si
para que não seja somente uma fase, mas algo que melhore para o futuro.
